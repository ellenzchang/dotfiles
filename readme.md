# dotfiles
Simply clone to home and run `setup.sh`

Setup iterm color scheme using `iterm/ayu.itermcolors`

Go into vim with `vim` and run
```bash
:source ~/.vimrc
```
Go into IntelliJ and run in within a file 
```bash
:source ~/.vimrc
```

## non-automatic configs
```bash
- RectangleConfig.json
```

# ai related setup
